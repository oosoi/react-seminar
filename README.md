## Getting Started

Use [fnm](https://github.com/Schniz/fnm) for Node.js version management.

```bash
fnm use
```

Use [pnpm](https://pnpm.io/) as a package manager.

```bash
pnpm install --frozen-lockfile
```

Run the development server:

```bash
pnpm dev
```
